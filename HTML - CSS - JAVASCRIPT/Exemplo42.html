Exemplo JavaScrip - Pilha

São estruturas de dados do tipo LIFO (last-in first-out), onde o último elemento a ser inserido, será o primeiro a ser retirado.

Em JavaScript usamos o push para inserir um elemento no topo da pilha e pop para remove-los

Podemos ainda usar outros métodos na pilha, por exemplo, podemos usar o métodos sort parra ordenar de forma crescente a nossa pilha.
